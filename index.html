Quick Start

Install pyDCMvn using pip:

pip install pyDCMvn

Basic usage example:

import pyDCMvn # Load a DICOM file dcm = pyDCMvn.load_dicom("path/to/file.dcm") # Process the data processed = pyDCMvn.process(dcm) # Save the result pyDCMvn.save_dicom(processed, "output.dcm")
