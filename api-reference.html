--- layout: default title: API Reference permalink: /api-reference/ --- # API Reference ## Core Functions ### `load_dicom(filepath)` Load a DICOM file from the specified path. **Parameters:** - `filepath` (str): Path to the DICOM file **Returns:** - `Dataset`: DICOM dataset object **Example:** ```python dataset = pyDCMvn.load_dicom('example.dcm') ``` ### `load_dicom_directory(directory_path)` Load all DICOM files from a directory. **Parameters:** - `directory_path` (str): Path to directory containing DICOM files **Returns:** - `List[Dataset]`: List of DICOM dataset objects **Example:** ```python datasets = pyDCMvn.load_dicom_directory('/path/to/dicom/files') ``` ### `save_dicom(dataset, filepath)` Save a DICOM dataset to a file. **Parameters:** - `dataset` (Dataset): DICOM dataset object - `filepath` (str): Output file path **Example:** ```python pyDCMvn.save_dicom(dataset, 'output.dcm') ``` ## Image Processing Functions ### `enhance_contrast(image_array, method='clahe')` Enhance image contrast using various methods. **Parameters:** - `image_array` (numpy.ndarray): Input image array - `method` (str): Enhancement method ('clahe', 'histogram', 'adaptive') **Returns:** - `numpy.ndarray`: Enhanced image array ### `apply_filter(image_array, filter_type='gaussian', **kwargs)` Apply various filters to medical images. **Parameters:** - `image_array` (numpy.ndarray): Input image array - `filter_type` (str): Type of filter ('gaussian', 'median', 'bilateral') - `**kwargs`: Additional filter parameters **Returns:** - `numpy.ndarray`: Filtered image array ### `normalize_image(image_array, method='minmax')` Normalize image pixel values. **Parameters:** - `image_array` (numpy.ndarray): Input image array - `method` (str): Normalization method ('minmax', 'zscore', 'percentile') **Returns:** - `numpy.ndarray`: Normalized image array ## Vietnamese Healthcare Functions ### `validate_vn_patient_id(patient_id)` Validate Vietnamese patient ID format. **Parameters:** - `patient_id` (str): Patient ID to validate **Returns:** - `bool`: True if valid, False otherwise ### `format_vn_patient_name(name)` Format patient name according to Vietnamese conventions. **Parameters:** - `name` (str): Patient name to format **Returns:** - `str`: Formatted patient name ### `encode_vietnamese_text(text)` Encode text for Vietnamese character support. **Parameters:** - `text` (str): Text to encode **Returns:** - `str`: Encoded text ## Utility Functions ### `get_dicom_info(dataset)` Extract key information from a DICOM dataset. **Parameters:** - `dataset` (Dataset): DICOM dataset object **Returns:** - `dict`: Dictionary containing key DICOM information ### `convert_pixel_data(dataset, output_format='numpy')` Convert DICOM pixel data to different formats. **Parameters:** - `dataset` (Dataset): DICOM dataset object - `output_format` (str): Output format ('numpy', 'pil', 'opencv') **Returns:** - Various: Converted pixel data in specified format ### `anonymize_dicom(dataset, fields=None)` Anonymize DICOM data by removing or replacing sensitive information. **Parameters:** - `dataset` (Dataset): DICOM dataset object - `fields` (list): List of fields to anonymize (default: standard set) **Returns:** - `Dataset`: Anonymized DICOM dataset ## Exception Classes ### `DicomReadError` Raised when unable to read a DICOM file. ### `InvalidFormatError` Raised when DICOM file format is invalid. ### `ProcessingError` Raised when image processing operations fail. ### `ValidationError` Raised when data validation fails. ## Constants ### `SUPPORTED_TRANSFER_SYNTAXES` List of supported DICOM transfer syntaxes. ### `VIETNAMESE_ENCODING` Default encoding for Vietnamese text in DICOM files. ### `DEFAULT_WINDOW_SETTINGS` Default window/level settings for different modalities.
